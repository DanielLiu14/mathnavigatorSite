# Advanced CSS

## Centering things Horizontally
(text only)
```
text-align: middle
```

```
display: block
width: ___px
margin: 0 auto
```

```
position: absolute
left: 50%
transform: translateX(-50%)
```

## Centering things Vertically
```
position: absolute
top: 50%
transform: translateY(-50%)
```

Only for aligning images with other images or text
```
display: inline-block
vertical-align: middle
```
